# 接口地址

请求方式：`GET`

返回格式：`JPEG`

```API
https://api.nxvav.cn/api/qqimg/
```

# 请求示例

[https://api.nxvav.cn/api/qqimg/?qq=123456](https://api.nxvav.cn/api/qqimg/?qq=123456)

# 使用场景

#### **html**

