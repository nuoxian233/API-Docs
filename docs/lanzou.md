# 请求参数

| 参数名 | 类型 | 示例 | 说明 |
| ----- | ---- | ---- | ---- |
| url | 必填 | 蓝奏外链链接 | 蓝奏外链链接 |

| 参数名 | 类型 | 示例 | 说明 |
| ----- | ---- | ---- | ---- |
| pwd | 参数一(可空) | 蓝奏外链密码 | 蓝奏外链密码 |

| 参数名 | 类型 | 示例 | 说明 |
| -----  | ---- | ---- | ----- |
| type |  参数二(可空) | 下载类型 | 下载类型 |
| down | 可空 | 直接下载文件 |  直接下载文件 |

# 返回参数

| 返回参数 | 说明 |
| ------- | ---- |
| code | 状态码 |
| data | 数据 |
| name | 文件名 |
| author | 分享者 |
| time | 分享时间 |
| size | 文件大小 |
| url | 文件直链 |

# 状态代码

| 返回状态 | 说明 |
| ------- | ---- |
| 200 | 解析成功 |
| 201 | 链接错误 |
| 202 | 密码错误/密码为空 |
